系統功能說明
帳號與權限管理
三種角色（ADMIN / MANAGER / EMPLOYEE），JWT 身份驗證，BCrypt 密碼加密，Spring Security 權限控管。
年度目標設定
MANAGER 設定年度 RE100 綠電目標比例與預估用電量，系統計算年度需求綠電量。
太陽能設備管理
管理企業太陽能設備清單，登記每月實際發電量，系統計算理論發電量與達標率。
供電合約管理
建立 CPPA 長約與售電業供電合約，管理合約狀態與費率，支援合約終止與到期追蹤。
